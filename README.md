## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn
- API keys (Google AI, OpenAI, etc.)

### Installation
```bash
git clone https://github.com/ZOOKA-AI/zooka-os-version-7.git
cd zooka-os-version-7
npm install
```

### Configuration
Create `.env` file:
```env
API_KEY=your_api_key_here
```

### Usage
```bash
npm start
```
